# 👟 Locker Rush - Foot Locker Promo Experience

## 📖 À propos du projet
"Locker Rush" est un mini-site promotionnel événementiel conçu pour la marque Foot Locker. Le concept central repose sur un jeu **100% gagnant** destiné à engager la communauté et à booster l'acquisition de clients. L'expérience web guide l'utilisateur à travers un tunnel de conversion gamifié : de la page d'accueil à la découverte du lot garanti, jusqu'au formulaire d'inscription final.

## 🚀 Fonctionnalités clés
* **Tunnel de conversion interactif :** Parcours utilisateur fluide, axé sur la gamification.
* **UI/UX Design moderne :** Implémentation d'un effet "glass-morphism" (transparence et flou) pour une interface esthétique et cohérente.
* **Responsive Design "Pixel Perfect" :** Interface entièrement adaptée aux mobiles. Testée rigoureusement sur les simulateurs iOS (Xcode) pour garantir le respect des *safe areas* et une expérience optimale sur tous les écrans, du petit iPhone SE aux modèles Pro Max.

## 🛠️ Technologies utilisées
* **HTML5** : Architecture et sémantique.
* **CSS3** : Flexbox, Media Queries, effet Glass-morphism, animations "from scratch".

## ⚙️ Déploiement et Lancement
Ce projet est construit sans framework externe. Pour le visualiser en local :
1. Clonez ce dépôt : `git clone [URL_DE_TON_REPO]`
2. Ouvrez simplement le fichier `accueil.html` dans votre navigateur web.
3. **Pour les tests mobiles :** Il est recommandé de lancer un serveur local (ex: `python3 -m http.server 8000`) pour visualiser le rendu exact sur un émulateur de smartphone.
